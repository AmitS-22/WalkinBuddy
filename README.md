# WalkinBuddy

WalkinBuddy is a Flutter-based Smart Hospitality and Safety application designed to help users discover hotels, book stays, access emergency SOS services, and navigate nearby locations using Google Maps.

---

## Features

- Firebase Authentication
  - Login
  - Signup
  - Forgot Password

- Hotel Browsing
  - Featured Hotels
  - Hotel Details
  - Search Hotels

- Booking System

- Google Maps Integration

- Live Location

- SOS Emergency Feature

- User Profile

- Reviews & Ratings

- Responsive UI

---

## Tech Stack

- Flutter
- Dart

## Getting Started

Clone the repository

```bash
git clone https://github.com/AmitS-22/WalkinBuddy.git
```

Move into the project

```bash
cd WalkinBuddy
```

Install dependencies

```bash
flutter pub get
```

Run the project

```bash
flutter run
```
